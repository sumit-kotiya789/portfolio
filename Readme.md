## ðŸŒŸ Key Interactive Features

### 1. Real-Time Blockchain Node Metrics
Integrated **Ethers.js v6** to bridge the gap between web technology and the **BNB Smart Chain (BSC)**.
* **Live Monitoring:** Direct JSON-RPC connection to track block numbers and transaction hashes in real-time.
* **Network Health:** Visualizes active peer counts and node status.
* **Live Market Data:** Real-time BNB/USDT price fetching via the Binance API.

### 2. Backend API Playground
A custom-built REST API simulator that allows recruiters to "test" my backend logic directly in the browser.
* **HTTP Simulations:** Supports `GET` requests with a focus on status code handling (`200 OK`, `404 Not Found`, `429 Too Many Requests`).
* **Rate Limiting:** Demonstrates an understanding of API security and resource management.

### 3. Interactive Shell (Bash Simulation)
A functional command-line interface built from scratch to provide a unique, developer-centric navigation experience.
* **CLI Commands:** Execute commands like `about`, `skills`, and `resume`.
* **Micro-Interactions:** Includes a typewriter greeting effect and smooth-scroll history logging.

### 4. System Architecture Visualization

* A visual breakdown showing the data flow between **Android Clients (Retrofit/MVVM)**, **Middleware (PHP/Node.js)**, and the **Blockchain Layer (Solidity/Ethers.js)**.
